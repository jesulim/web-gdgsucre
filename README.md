## 📖 Documentación
Accede a la documentación técnica en: **[docs.gdgsucre.com](https://docs.gdgsucre.com/)**

## 🚀 Inicio Rápido
```bash
bun install
bun dev
```

## 🧞 Comandos
| Comando | Acción |
| :--- | :--- |
| `bun dev` | Inicia el servidor de desarrollo |
| `bun run build` | Compila para producción |
| `bunx biome check --write` | Ejecuta Lint & Format |
